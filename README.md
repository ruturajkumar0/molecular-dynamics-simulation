# Parkinson_MD

Molecular Dynamics Simulation of Acetylcholinesterase (AChE) with Natural Flavonoids.

## Ligands
- Quercetin
- Luteolin
- Kaempferol
- Apigenin

## Software Used
- GROMACS 2026.2
- AutoDock Vina
- PyMOL
- Discovery Studio Visualizer

## Workflow

Protein Preparation
→ Molecular Docking
→ Complex Generation
→ Solvation
→ Energy Minimization
→ NVT Equilibration
→ NPT Equilibration
→ Production MD
→ Trajectory Analysis
